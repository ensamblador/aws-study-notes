# Dynamo DB

## DynamoDB Streams

### Kinesis Adapter

La forma recomendad de leer streams es utilizar el Kinesis Adapter, esto permite utilizar el mismo KCL para consumir los streams (como si fuese un Kinesis Stream)

### Encriptación

La data en descanso también se encripta

### Duración

Solo duran 24 Horas
